> 简单和规律确实能带来放松


# 安装gdb  
ubuntu环境下用apt下载，centos只需要把apt替换成yum即可  
````
sudo apt update
sudo apt install gdb
````

# 编译准备  
正常编译，不带有gdb调试信息  
````
g++ test.cpp -o test
````

加入gdb调试信息
````
g++ -g test.cpp -o test
````
